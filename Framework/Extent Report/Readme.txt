• Integrate ExtentReport with Eclipse
	Create new project
	Select Maven project
	Select maven-archetype-quickstart
	Provide Group id, Artifact id, package name.
	Write test cases in (src/test/java) and javafile name must end by Test name, like SeleniumTest.java, etc.

• Understanding POM.xml file dependencies..
	Add dependencies in pom.xml from Maven Repository
	Link -> https://mvnrepository.com/

----------------------------

• How to find the dependencies of the Softwares?
	Search for dependencies which is required like ExtentReports, TestNG, Seleniumjava, etc.
	Link -> https://mvnrepository.com/

	ExtentReports 	    : Used to create report file
